# Legion Chain Pro - Telegram Bot

Телеграм-бот для автоматизации Legion Chain Score с поддержкой системы подписок на Telegram Stars.

## Возможности

- ✅ Три типа подписок (STARTER, PROFESSIONAL, ENTERPRISE)
- ✅ Оплата через Telegram Stars
- ✅ Поддержка 3 блокчейнов (Arbitrum, Solana, Sui)
- ✅ Админ-панель
- ✅ История транзакций
- ✅ Telegram Mini App интеграция

## Цены

- STARTER: 700⭐ (7 дней)
- PROFESSIONAL: 1000⭐ (14 дней)
- ENTERPRISE: 1500⭐ (30 дней)
